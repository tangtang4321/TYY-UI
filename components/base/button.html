tyy-button #

按钮分为 primary - 主要按钮、secondary - 次要按钮（默认）、dashed - 虚线按钮、outline - 线形按钮、plain - 朴素按钮、text - 蓝色文本按钮、textPlain - 黑色文本按钮七种类型。

<template>
  <tyy-space>
    <tyy-button>primary</tyy-button>
    <tyy-button type="secondary">secondary</tyy-button>
    <tyy-button type="dashed">dashed</tyy-button>
    <tyy-button type="text">text</tyy-button>
    <tyy-button type="outline">outline</tyy-button>
    <tyy-button type="plain">plain</tyy-button>
    <tyy-button type="plainText">plaintext</tyy-button>
  </tyy-space>
</template>
<script lang="ts" setup></script>

按钮分为 mini、small、medium、large 四种尺寸.通过设置 long 属性，使按钮的宽度跟随容器的宽度。

<template>
  <tyy-space direction="vertical" fill>
    <tyy-space>
      <tyy-button size="mini"> mini </tyy-button>
      <tyy-button size="small"> small </tyy-button>
      <tyy-button size="medium"> medium </tyy-button>
      <tyy-button size="large"> large </tyy-button>
    </tyy-space>
    <tyy-button long> long </tyy-button>
  </tyy-space>
</template>
显示代码

按钮的状态分为 normal - 正常(默认)、warning - 警告、danger - 危险 3 种，可以与按钮类型同时使用。另外还有禁用状态和加载状态

<template>
  <tyy-space>
    <tyy-button disabled>disabled</tyy-button>
    <tyy-button loading disabled>loading</tyy-button>
    <tyy-button status="normal">normal</tyy-button>
    <tyy-button status="warning">warning</tyy-button>
    <tyy-button status="danger">danger</tyy-button>
  </tyy-space>
</template>
显示代码

按钮可以嵌入图标。在只设置图标时，按钮的宽高相等。

<template>
  <tyy-space>
    <tyy-button>
      <template #icon> <icon-plus /> </template>
    </tyy-button>
    <tyy-button shape="round">
      <template #icon> <icon-delete /> </template>
    </tyy-button>
    <tyy-button type="outline"> <icon-caret-left />上一页 </tyy-button>
    <tyy-button type="outline" shape="round">
      下一页
      <icon-caret-right />
    </tyy-button>
    <tyy-button type="text">
      <template #icon> <icon-download /> </template>
    </tyy-button>
    <tyy-button type="text">
      <template #icon> <icon-filter /> </template>
    </tyy-button>
  </tyy-space>
</template>
<script lang="ts" setup>
import {
  IconPlus,
  IconDelete,
  IconCaretRight,
  IconCaretLeft,

  IconFilter,
} from "@arco-design/web-vue/es/icon";
</script>
